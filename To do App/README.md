# To-Do App with Login

## Project Overview
This project is a simple To-Do application that includes user authentication. Users can log in to manage their tasks effectively.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- User login
- Add, edit, and delete tasks
- Mark tasks as complete

## Getting Started
1. Clone the repository.
2. Open `index.html` in your browser.

## Usage
- Enter your credentials to log in.
- Use the interface to manage your tasks.
