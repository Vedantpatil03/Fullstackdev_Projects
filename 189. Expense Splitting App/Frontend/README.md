# Expense Splitting App - Frontend

Frontend application for Expense Splitting App

## Project Structure

```
├── index.html          # Main HTML file
├── styles/
│   └── main.css       # Main stylesheet
├── js/
│   └── app.js         # Main JavaScript file
└── assets/            # Images, fonts, etc.
```

## Features

- Responsive Design
- Modern CSS
- Vanilla JavaScript

## Getting Started

### Installation

```bash
npm install
```

### Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
```

## API Configuration

Update `js/app.js` with your backend API URL:

```javascript
const API_BASE_URL = 'http://localhost:3000/api';
```
