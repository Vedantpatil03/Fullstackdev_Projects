# Event Registration Website

This project is a simple event registration website built using HTML, CSS, and JavaScript. Users can register for events by filling out a form, and they will receive a confirmation message upon submission.

## Project Structure
- `index.html`: The main HTML file containing the structure of the website.
- `styles.css`: The CSS file for styling the website.
- `script.js`: The JavaScript file for handling form submission and displaying confirmation messages.

## How to Run
1. Open `index.html` in a web browser.
2. Fill out the registration form and submit.
3. A confirmation message will be displayed.

## Technologies Used
- HTML
- CSS
- JavaScript
